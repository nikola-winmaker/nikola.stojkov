## Conference Paper Presentation

This is an interactive presentation of the research paper **"XCC: A Universal Communication Interface Composer for Heterogeneous Systems"** presented at the 30th International Conference on Information Technology (IT) in Žabljak, February 24-28, 2026.

### Paper Overview

XCC presents a lightweight communication composition mechanism based on a domain-specific language for designing compile-time known communication interfaces and protocols in heterogeneous embedded systems.

**Key Contributions:**
- Architectural connectivity specification
- Interface contracts definition
- Communication profiles capturing
- Practical validation and code generation
